What is HTML-Render?

HTML-Render is an API that allows you to render HTML and CSS code to get a physical representation of the DOM, either from your own web page or from an external web site. It is very easy and simple to use and the quality of the generated images is very good.

How to use?

GitHub

If you want to contribute to the project you can do it through the GitHub repository, where you will find the source code of the API. Some ways in which you can help are by reporting bugs, resolving bugs or simply adding new features to the API. Remember to review the README.md file, as it contains the steps to follow so that your proposal is not rejected.
