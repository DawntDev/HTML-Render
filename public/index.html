What is HTML-Render?

HTML-Render is an API that allows you to render HTML and CSS code to get a physical representation of the DOM, either from your own web page or from an external web site. It is very easy and simple to use and the quality of the generated images is very good.

How to use HTML-Render?

From the browser

Select a method to use

Go to the HTML-Render page, and select the method you want to use. The first one is to render your own code and the second one is to capture an existing page.

Customize your render

Fill in the image customization fields, as well as the element and format you want to obtain. For more information check the documentation.

Await for the render to finish

Wait for HTML-Render to do its job, it may take between 2 and 5 seconds to perform the operation, depending on the speed of your internet.

The body of the request

The body of the request is composed of a JSON object, which varies according to the method you want to use. Although in essence they are very similar. In the following examples you will be able to see which are the fields of which the body of your request is composed.

{
elements "<h1>Hello World!</h1>" ,
css "h1 { color: red; }" ,
js "console.log('Hello World')" ,
selector "body" ,
format "png" ,
size [ "1920" , "1080" ] ,
timeout 2.5
}
{
url "https://www.google.com" ,
selector "body" ,
format "png" ,
size [ "1920" , "1080" ] ,
timeout 2.5
}

/api/v1/render

Using this method you will be able to render your own code, and get the render in the format that best suits your needs.

Methods

POST

Key Description Type Example
elements The elements that you want to render string "<h1>Hello World!</h1>"
css The CSS that you want to render string "h1 { color: red; }"
js The JavaScript that you want to render string "console.log('Hello World')"
selector The CSS selector of the element you want to render string "body > div.component"
format The format of the image you want to render string Only one "png", "jpg", "bin", "base64"
size The size of the image you want to render array[ string , string ] [ "1920" , "1080" ]
timeout The timeout of the request float 2.5

/api/v1/urlToImage

With this method you can render web pages live. This through the URL of the page, and thus obtain a representation in the format you need.

Errors

During the applications you may encounter some answers that are not as expected.Therefore, through this list you will find some of the most common mistakes. So that you can resolve them as soon as possible.

Error Description
Missing key In the body of your request is missing some key that is necessary for the rendering of your request.
Invalid type for key The key value is not of the correct type, so your request cannot be accepted for rendering.
The key is empty You have provided a key with a value but its value is empty.
Invalid format The format of the image you want to render is not valid.
Error while rendering the page Some internal error has occurred while trying to render your request. This kind of errors can be considered as bugs, so please report them.
